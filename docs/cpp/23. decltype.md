# decltype

:::note
Since C++ 11
:::

```cpp
const int &get_value();

int main() {
  decltype(get_value()) j = 2;
  static_assert(std::is_same<decltype(get_value()), const int &>::value, "TYPE SHOULD BE THE SAME");
  std::cout << j;
}
```

`decltype(get_value()) j = 2;` 中，get_value 不会执行，而是通过编译器分析，以获得 `get_value` 函数的返回值的类型，从而声明 j 的类型。
